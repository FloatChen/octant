Package octant :: Module io

Module io

source code

A wrapper for netCDF4.Dataset and netCDF4.MFDataset

possible uses include:

with an input of a string:
nc = pyroms.Dataset(file) # returns netCDF4.Dataset object based on file nc = pyroms.MFDataset(file) # returns MFnetCDF4.Dataset object based on file (with wildcard chars)
with an input of a list of files:
nc = pyroms.Dataset(files) # returns MFnetCDF4.Dataset object based on list of files nc = pyroms.MFDataset(files) # returns MFnetCDF4.Dataset object based on list of files
with an input of a netCDF4.Dataset or MFnetCDF4.Dataset object:
nc = pyroms.Dataset(nc) # passes through netCDF4.Dataset or MFnetCDF4.Dataset object nc = pyroms.MFDataset(nc) # passes through MFnetCDF4.Dataset object based on file (with wildcard chars)
