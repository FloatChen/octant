        BoundaryInteractor

bry = BoundaryClick(x=[], y=[], beta=None, ax=gca(), **gridgen_options)

If x, y and beta are not given, and interactive polygon creation session is started. The initial boundary is sketched out by clicking the points (in a counterclockwise manner, usually starting in the upper lefthand corner of the boundary). At this point the verticies are marked by orange circles.

Switch to editing mode by hitting return. This changes the vertecies to black. At this point, beta values may be created to define the corners of the grid (see below). Current data are always available in bry.x, bry.y (combined in p.verts) and bry.beta attributes.

Key commands:

enter : switch to grid editing mode

t : toggle visibility of verticies d : delete a vertex i : insert a vertex at a point on the polygon line

p : define vertex as beta=1 (a Positive turn, marked with green triangle) m : define vertex as beta=1 (a Negative turn, marked with red triangle) z : define vertex as beta=0 (no corner, marked with a black dot)

G : generate grid from the current boundary using gridgen T : toggle visability of the current grid
