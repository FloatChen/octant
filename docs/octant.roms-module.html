nc_gls_dissipation(nc, tidx)

source code 

Return the dissipation, based on tke, gls and the gls scheme parameters cmu0, m, n, and p. Usage:

eps = nc_gls_dissipation(nc, tidx)

where nc is a netcdf object or filename, tidx is the time index to return

nc_grid(nc)

source code 

Return grid class based on ROMS GRD file, or other output file with grid information. The NetCDF file must contain either 'vert' veriables, or the verticies will be calculated with 'rho' and angle points.
