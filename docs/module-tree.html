Module Hierarchy

  • octant: OCTANT is a toolkit for working with ocean models on a C-Grid
    • octant.cf: cf.py - classes around CF compliant files
    • octant.depths: Various vertical coordinates
    • octant.extern: External packages
      • octant.extern.delaunay: Delaunay triangulation and interpolation tools.
        • octant.extern.delaunay.interpolate
        • octant.extern.delaunay.setup
        • octant.extern.delaunay.testfuncs: Some test functions for bivariate interpolation.
        • octant.extern.delaunay.triangulate
      • octant.extern.greatcircle
      • octant.extern.poly_editor: This is an example to show how to build cross-GUI applications using matplotlib event handling to interact with objects on the canvas
      • octant.extern.polygeom: Polygon geometry.
      • octant.extern.pupynere: NetCDF reader.
    • octant.getm: octant/getm - octant tools for GETM
    • octant.grid: Tools for creating and working with Arikawa C-Grids
    • octant.io: A wrapper for netCDF4.Dataset and netCDF4.MFDataset
    • octant.ocean: Model agnostic ocean tools
      • octant.ocean.depths: depths.py
      • octant.ocean.state: Tools for working with oceanic data
      • octant.ocean.step3d_t: step3d_t.py
      • octant.ocean.time
      • octant.ocean.velocity: velocity class.py
    • octant.roms: Tools specific for the Regional Ocean Modeling System
    • octant.seawater: Tools for working with seawater data.
    • octant.slice: Tools for slicing model fields.
    • octant.sw: Seawater -- Python functions for properties of sea water Bjørn Ådlandsvik <bjorn@imr.no>, Institute of Marine Research, Version 1.1, 13 November 2002
      • octant.sw.density: Seawater density module
      • octant.sw.heat: Seawater heat module
      • octant.sw.misc: Miscellaneous sea water functions
      • octant.sw.salinity: Seawater salinity module, providing salt and cond functions
      • octant.sw.test
    • octant.tools: Tools for working with ROMS model input and output.
