Bdyinfo(filename): handle for GETM bdyinfo file/definition

The Bdyinfo returns an object connected to a GETM bdyinfo file (usually bdyinfo.dat). It basically reads in the bdyinfo file specified with the filename and knows the list of indices in the right order for reading/writing Grids and Datasets
