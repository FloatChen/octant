octant/getm - octant tools for GETM

The getm module includes some very specific tools for the General Estuarine Transport Model (www.getm.eu).

Included classes:

Bdyinfo - handle for a bdyinfo file object Bdy2d - handle for creating and filling of a bdy_2d.nc file Bdy3d - handle for creating and filling of a bdy_3d.nc file

Dependencies:

netCDF4 (as it is now, the only included netCDF library) numpy
