 3  OCTANT is a toolkit for working with ocean models on a C-Grid

 5  Octant (Ocean C-grid model seTup and Analysis Toolkit) is based on the 
 6  python/numpy/matplotlib scientific python suite. NetCDF I/O is based on the 
 7  NetCDF4-python package. The toolkit contains general modeling tools for dealing 
 8  with arrays, diagnosing standard properties, curvilinear grid generation, and 
 9  interpolation. Specific tools are also included for ROMS and GETM.
